# Rules Implementation Plan

## [Medium] Phase 1 - Priority 2: abutment criteria for (pmos/nmos, different vt, different length/width) for both same row or differe...
**Full Feature:** abutment criteria for (pmos/nmos, different vt, different length/width) for both same row or different rows

### Technical Implementation Steps:
1. **Parser Development:** Build a parser for tech-dependent rule files (JSON/YAML/XML).
2. **Rule Mapping:** Translate technology parameters (min space, abutment conditions) into logical constraints.
3. **Engine Implementation:** Develop the check/enforcement logic for diffusion breaks, VT boundaries, and layer spacing.
4. **API Exposure:** Create an interface for the placement and flow engines to query rules dynamically.
