# Placement Implementation Plan

## [Hard] Phase 1 - Priority 1: Matching pattern to consider, commom centroid for CM, interdigitation for diff pair or diff branches...
**Full Feature:** Matching pattern to consider, commom centroid for CM, interdigitation for diff pair or diff branches

### Technical Implementation Steps:
1. **Geometric Modeling:** Define the bounding boxes and relative coordinates for the devices/groups.
2. **Constraint Engine:** Implement algorithms for symmetry, matching (CC, interdigitation), and overlap detection.
3. **Optimization:** Apply area-shaping and placement heuristics to minimize parasitics and area.
4. **Tech-Rule Integration:** Validate placement against technology-specific spacing and abutment rules.
5. **Verification:** Compare results against manual golden layouts for density and signal integrity.
