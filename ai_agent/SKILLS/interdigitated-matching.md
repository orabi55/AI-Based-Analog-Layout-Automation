# Interdigitated Matching

Description: Interleave fingers from matched devices using deterministic proportional distribution to improve matching and routing regularity without changing device inventory.

When to apply:
- User intent includes interdigitated, alternate, ABAB, or ratio-based mixing.
- Strategy text calls for interleaving matched fingers in shared rows.

Core guidance:
- Process all IG devices in the same row as one unified group.
- Distribute fingers according to ratio targets, not ad-hoc swapping.
- Avoid terminal clustering; maintain spread across the row.
- Enforce one-slot-per-device-instance and preserve all IDs/fingers.
